# Ansible `stat` Module

The **stat module** is used to get information about files or directories on a target machine (managed node).  
It does **NOT modify anything** — it only reads file/directory information.

## Details Provided by `stat`

- **stat.exists** → file exists  
- **stat.isdir** → is directory  
- **stat.mode** → file permissions  
- **stat.size** → file size
